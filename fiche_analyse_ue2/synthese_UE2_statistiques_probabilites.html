1. Les types de variables
Type de variable Définition Exemples Description
Quantitative continue Nombre infini de valeurs possibles Taille (1.75m), poids (68.3kg), glycémie (1.2g/L) Position + Dispersion
Quantitative discrète Nombre fini de valeurs possibles Nb d'enfants (0, 1, 2, 3...), nb de consultations (1, 2, 3...) Position + Dispersion
Qualitative nominale Catégories sans ordre Couleur des yeux (bleu, vert, marron), groupe sanguin (A, B, AB, O) Effectif + %
Qualitative ordinale Catégories avec ordre Jamais / rarement / parfois / souvent / toujours Effectif + %
Dichotomique 2 modalités uniquement Oui/non, homme/femme, fumeur/non-fumeur, malade/sain Effectif + %

A retenir

Variable ordinale = PLUS d'information qu'une dichotomique, mais MOINS qu'une quantitative continue.

Discrétisation = passage d'une variable continue à une variable discontinue (ex : 2.01m → 2m)

Mesures de POSITION Où ?

Pièges classiques

  • Le 9ème quartile n'existe pas (seulement Q1, Q2, Q3)
  • L'intervalle interquartile = DISPERSION, pas position
  • Un quartile seul (Q1, Q3) = POSITION

Piège classique

Stratifié (probabiliste) ≠ Quotas (non probabiliste)

Les deux utilisent des sous-groupes, mais seul le stratifié utilise le tirage aléatoire.

4. Intervalle de confiance
Situation Conséquence
Échantillon plus GRAND IC plus ÉTROIT = précision MEILLEURE
Échantillon plus PETIT IC plus LARGE = précision MOINS BONNE
Risque alpha augmente IC se rétrécit
IC à 95% 95% de chances dedans, 5% dehors
Données EXHAUSTIVES PAS d'IC car pas d'estimation

Formules

IC = 1 - alpha

Intervalle de confiance : Échantillon → Population (estimation)

Intervalle de Pari : Population → Échantillon

Piège classique

Quand on a un registre exhaustif (élections, registre des dialysés...), il n'y a pas d'échantillon donc pas d'IC.

C'est de la statistique descriptive, pas inférentielle.

5. Tests d'hypothèse
Élément Définition
H0 (hypothèse nulle) Pas de différence entre les groupes
H1 (hypothèse alternative) Il existe une différence entre les groupes
Risque alpha (1ère espèce) Rejeter H0 à tort = faux positif
Risque beta (2ème espèce) Ne pas rejeter H0 à tort = faux négatif
Puissance 1 - beta = capacité à détecter une vraie différence
p (petit p) Risque de se tromper si on rejette H0
Règle de décision
Si... Alors... Interprétation
p < alpha On rejette H0 Différence SIGNIFICATIVE
p > alpha On ne rejette pas H0 Différence NON significative
p très élevé (0.87, 0.95) On ne rejette pas H0 Vraiment pas de différence
p proche du seuil (0.06, 0.07) On ne rejette pas H0 Possiblement manque de puissance

A retenir

  • Alpha et beta sont ANTAGONISTES
  • Le seuil alpha se définit A PRIORI (avant le test)
  • Les tests sont basés sur un raisonnement par l'absurde
  • H0 et H1 sont mutuellement exclusives
6. Quel test pour quelle situation ?
Corrélation entre 2 variables quantitatives
Distribution Coefficient
Normale Pearson
Non normale Spearman

Coefficient de corrélation

  • Compris entre -1 et +1
  • Corrélation ≠ causalité
  • Coef < 0 : relation inverse
  • Coef > 0 : relation positive
7. Tests paramétriques vs non paramétriques
Caractéristique PARAMÉTRIQUES NON PARAMÉTRIQUES
Exemples Student, Test Z, Pearson Mann-Whitney, Wilcoxon, Spearman
Puissance PLUS puissants MOINS puissants
Robustesse MOINS robustes PLUS robustes
Conditions Distribution normale requise Aucune condition
Fonctionnement Utilise les valeurs Utilise les rangs
Valeurs extrêmes Sensibles Moins sensibles

Piège classique

Ne pas confondre puissance et robustesse :

  • Paramétrique = plus puissant, moins robuste
  • Non paramétrique = moins puissant, plus robuste

9. Biais et randomisation
Type de biais Définition Comment le minimiser
Biais de sélection Erreur dans l'identification de la population Définir clairement la population
Biais d'information Erreur dans la mesure Standardiser les instruments

A retenir

Un biais est une erreur SYSTÉMATIQUE, il n'est PAS corrigé par l'augmentation de la taille d'échantillon.

La randomisation
PERMET de...
Équilibrer les facteurs de confusion
Répartir facteurs connus ET inconnus
NE PERMET PAS de...
Aveugler investigateur et patient
Neutraliser les biais de sélection
Augmenter la puissance
Éviter les perdus de vue
