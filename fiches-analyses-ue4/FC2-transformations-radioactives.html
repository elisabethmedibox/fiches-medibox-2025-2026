Recommandations Niveau 2 (ciblage de parties du cours)

1) Les modes de désintégration (α, β+, β−, CE)

Objectif : Savoir associer chaque type de désintégration à son contexte nucléaire.

Désintégration α - noyaux lourds (Z > 82) ou avec trop de nucléons
Émission d'une particule α (noyau d'hélium). A diminue de 4, Z diminue de 2.

Désintégration β− - excès de neutrons
Transformation : n → p + e− + antineutrino. A constant, Z +1.

Désintégration β+ - excès de protons
Transformation : p → n + e+ + neutrino. A constant, Z -1. Le positon s'annihile → 2 photons de 511 keV.

Capture électronique (CE) - en compétition avec β+ (pas β− !)
p + e− → n + neutrino

Tombé 5 ans / 5

2) Période physique vs biologique vs effective

Objectif : Ne jamais confondre les 3 types de périodes.

Période physique (Tp)
Temps pour que la moitié des noyaux se désintègrent. = période radioactive = demi-vie. Propriété intrinsèque du radionucléide.

Période biologique (Tb)
Temps pour que l'organisme élimine la moitié d'une substance. Ne dépend pas de la radioactivité.

Période effective (Teff)
Temps pour que la radioactivité dans l'organisme diminue de moitié. Combine les deux. Formule : 1/Teff = 1/Tp + 1/Tb. Toujours inférieure aux deux autres.

Tombé 3 ans / 5

3) Calculs de période et décroissance

Objectif : Savoir calculer rapidement sans calculatrice.

Ce qu'il faut retenir :

1 période → activité ÷ 2

2 périodes → activité ÷ 4

3 périodes → activité ÷ 8

Méthode QCM : on te donne un facteur (2, 4, 8...), tu comptes combien de périodes ça représente, puis tu divises le temps par ce nombre.

→ Voir accordéon 4 pour les exemples concrets

Tombé 2 ans / 5

4) Émissions accompagnant les désintégrations

Objectif : Associer chaque désintégration à ses émissions.

β− → électron + antineutrino (ν̄)

β+ → positon + neutrino (ν)

Spectre β → spectre CONTINU (pas de raies). L'énergie est partagée aléatoirement entre β et neutrino.

Annihilation β+ → e+ + e− = 2 photons γ de 511 keV (à 180°)

Tombé 2 ans / 5

Thèmes jamais tombés (à surveiller)

- Émission γ et conversion interne (désexcitation)

- Chaînes de désintégration

- Filiation radioactive (équilibre séculaire, transitoire)

Ces thèmes n'ont pas été interrogés sur les 5 dernières années mais font partie du programme.

Recommandations Niveau 1 (stratégie annales)

Objectif Niveau 1 : tu n'as pas le temps d'ouvrir le cours. Tu vas travailler uniquement les annales. Voici les 3 types de QCM qui reviennent et comment les maitriser.

TYPE 1 - Définition période physique vs effective

Tombé en 2021-22, 2022-23, 2024-25 - QCM quasi-identique
Comment ça tombe :
On te donne une phrase du type "le temps nécessaire pour que la radioactivité diminue de moitié dans l'organisme correspond à..." et tu dois choisir entre période physique, biologique, effective.
Ce que tu dois retenir :

Règle simple pour trancher :

1. Si on parle de radioactivité + dans l'organisme = période effective

(c'est le cas le plus fréquent aux QCM)

2. Si on parle de noyaux qui se désintègrent (sans mention de l'organisme) = période physique

3. Si on parle d'élimination par l'organisme (sans mention de radioactivité) = période biologique

Astuce : dès que tu vois "organisme" + "radioactivité" ensemble, c'est effectif. La période physique ne concerne QUE les noyaux, pas l'organisme.

⚠️ Le piège qui tombe à chaque fois :
"La période physique est toujours supérieure à la période biologique" → FAUX
Ça dépend du radionucléide ! Pas de règle générale.

TYPE 2 - Modes de désintégration

Tombé CHAQUE ANNÉE depuis 2020 - 5 ans / 5
Comment ça tombe :
Des phrases du type "La désintégration β+ est le mode que subissent les noyaux lourds" ou "Lors d'une β−, un proton devient neutron".
Apprends ce tableau par coeur :
Type Quand ? Transformation
α Noyaux LOURDS (trop de nucléons) Expulse 2p + 2n
β− Trop de NEUTRONS n → p (neutron devient proton)
β+ Trop de PROTONS p → n (proton devient neutron)
CE En compétition avec β+ Capture un e−
⚠️ Les 3 pièges qui tombent TOUT LE TEMPS :
  • "β+ pour les noyaux lourds" → FAUX (c'est α)
  • "β− : proton devient neutron" → FAUX (c'est l'inverse : n → p)
  • "CE en compétition avec β−" → FAUX (c'est avec β+)
Moyen mnémotechnique :

β− = le signe MOINS (-) → on ENLÈVE un neutron → n devient p

β+ = le signe PLUS (+) → on AJOUTE un neutron → p devient n

TYPE 3 - Calculs période / décroissance

Tombé en 2020-21 et 2023-24 - Même énoncé, chiffres différents
Comment ça tombe :
Version 1 : "L'activité diminue d'un facteur 8 en 6 jours. Quelle est la période ?"
Version 2 : "La période est de 2 jours. De quel facteur l'activité diminue-t-elle en 6 jours ?"
Méthode pas à pas :

Étape 1 : Apprends la table des puissances de 2

  • 21 = 2 → facteur 2 = 1 période
  • 22 = 4 → facteur 4 = 2 périodes
  • 23 = 8 → facteur 8 = 3 périodes
  • 24 = 16 → facteur 16 = 4 périodes

Étape 2 : Traduis l'énoncé

  • "Facteur 8" → regarde dans ta table → c'est 23 → donc 3 périodes

Étape 3 : Calcule

  • Si on cherche T : T = temps ÷ nombre de périodes
  • Exemple : 6 jours ÷ 3 périodes = 2 jours
Formule (pour info) :

A(t) = A0 × (1/2)t/T

En pratique, tu n'auras pas à l'utiliser. La méthode ci-dessus avec la table des puissances de 2 suffit.

Ce type de calcul doit être maitrisé :
C'est toujours le même raisonnement avec des chiffres différents. Si tu sais faire celui-ci, tu sauras faire tous les autres.

TYPE 4 - Calcul d'activité A = λ × N

Tombé en 2021-22 - QCM isolé mais peut revenir
Comment ça tombe :
On te donne un nombre de noyaux N et une période T. Tu dois calculer l'activité A.
Exemple 2021-22 : "On administre 3,7×1015 noyaux d'iode 131 (T = 8 jours ≈ 7×105 s, ln2 ≈ 0,7). L'activité est d'environ ?"
Méthode pas à pas :

Formule : A = λ × N = (ln2 / T) × N

Calcul :

  • λ = 0,7 / (7×105) = 10-6 s-1
  • A = 10-6 × 3,7×1015 = 3,7×109 Bq
  • = 3700 MBq (réponse D)

Astuce : les valeurs sont toujours choisies pour que le calcul tombe juste. Cherche les simplifications (ici 0,7/7 = 0,1 = 10-1).

Points de vigilance

⚠️ Les pièges ci-dessous sont issus des annales des 5 dernières années.

Piège tombé : "CE en compétition avec β−"

Années : 2020-21 Q1D, 2021-22 Q2E
Réponse : FAUX. La capture électronique est en compétition avec β+ (pas β−). Les deux concernent les noyaux avec excès de protons.

Piège tombé : "β− = transformation p en n"

Années : 2021-22 Q2D, 2023-24 Q2B
Réponse : FAUX. C'est l'inverse ! β− = n → p (neutron devient proton). β+ = p → n.

Piège tombé : "β+ = noyaux lourds"

Années : 2021-22 Q2C, 2022-23 Q2E, 2023-24 Q2A
Réponse : FAUX. β+ concerne les noyaux avec excès de PROTONS. Les noyaux lourds → désintégration α.

Piège tombé : "Spectre β = raies"

Année : 2023-24 Q2E
Réponse : FAUX. Le spectre β est CONTINU (pas de raies). L'énergie est partagée aléatoirement entre la particule β et le neutrino.

Piège tombé : "Tp toujours > Tb"

Année : 2022-23 Q3C
Réponse : FAUX. La période physique n'est PAS toujours supérieure à la période biologique. Ça dépend du radionucléide et de l'élément chimique.

Piège possible (non tombé mais plausible) :

Confusion entre "période effective = période physique" ou "période effective = période biologique". La période effective est TOUJOURS inférieure aux deux autres (elle combine les deux phénomènes).

Tu dois maîtriser l'ensemble du chapitre : tous les modes de désintégration (α, β+, β−, CE, émission γ, conversion interne), la cinétique complète (constante λ, loi de décroissance, calculs d'activité), et les notions de filiation radioactive.

À noter : L'émission γ, la conversion interne et les chaînes de désintégration n'ont pas été interrogées ces 5 dernières années - mais elles font partie du programme et pourraient tomber.

Concentre-toi sur les 4 thèmes récurrents :

• Période physique vs biologique vs effective - tombé 3/5 ans
• Les calculs période/décroissance - tombé 2/5 ans
• Les émissions accompagnant les désintégrations (neutrino, spectre continu) - tombé 2/5 ans

Survole le reste sans t'attarder. Tu peux faire l'impasse sur l'émission γ et la conversion interne (jamais tombées).

Niveau 1 - tu travailles quasi exclusivement sur les annales

Mon avis basé sur les statistiques :

Stratégie Niveau 1 : viable
Avec un SRI estimé autour de 60% et des QCM de calcul qui reviennent avec la même méthode (seuls les chiffres changent), ce chapitre est adapté à une stratégie annales.

Concrètement : Il y a 3 types de QCM qui reviennent chaque année. Les QCM de calcul (période/décroissance) utilisent strictement la même méthode - seules les valeurs numériques changent. Si tu maîtrises la méthode sur un QCM, tu maîtrises tous les QCM de ce type. Ça vaut donc le coup de les travailler à fond.

Attention toutefois : Ce chapitre est une base pour comprendre la suite de l'UE4 (production de radionucléides, imagerie, radioprotection). Si tu fais l'impasse sur le cours, tu risques de galérer sur les chapitres suivants. À toi de voir si tu peux te le permettre en fonction de ta situation.
